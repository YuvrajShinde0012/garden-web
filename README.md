# Garden-Web

Garden-Web is a MERN stack web app where users can order plants, fertilizers, and other gardening products online.

## ðŸš€ How to Run the Project

### 1. Clone the Project
```bash
git clone https://github.com/your-username/garden-web.git
cd garden-web
````

### 2. Install Dependencies

#### Backend

```bash
cd server
npm install
```

#### Frontend

```bash
cd ../client
npm install
```

### 3. Add Environment Variables

Create a file named `.env` inside the **server** folder:

```
MONGO_URI=your_mongo_database_url
JWT_SECRET=your_secret_key
PORT=5000
```

### 4. Start the Project

#### Start Backend

```bash
cd server
npm run dev
```

#### Start Frontend

```bash
cd client
npm start
```

---

Your project will now run on:

* **Frontend:** [http://localhost:3000](http://localhost:3000)
* **Backend:** [http://localhost:5000](http://localhost:5000)
